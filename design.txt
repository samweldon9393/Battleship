util.py
    - Data Classes
        - Coordinate: (row: int, col: int)
            + rows: {LETTER -> INDEX}, inds: {INDEX -> LETTER}, hash()
        - AttackResult: hit: bool, sunk: bool, ship: Ship | None
    - Enums
        - CellState: UNKNOWN, MISS, HIT, SUNK, INVALID
        - Orientation: HORIZONTAL, VERTICAL
        - Difficulty: EASY, MEDIUM, HARD
        - Modes: HUNT, TARGET
    - Constants
        - MAX_SHIP_SIZE
        - TOTAL_SHIPS
        - BOARD_SIZE

/*****************************************************************************/
 *                       Classes, Members, APIS                              *
/*****************************************************************************/

- Ship  ← base class
    * sid:  int ← unique identifier for hashing
    * name: str
    * size: int
    * hits: int
    * occupied: list
    % place() -> bool
    % is_sunk() -> bool
    % occupies(coord) -> bool
    - Derived Classes
        - AircraftCarrier(Ship) size=5
        - Battleship(Ship)      size=4
        - Cruiser(Ship)         size=3
        - Destroyer(Ship)       size=2
        - Submarine(Ship)       size=1

- Board
    """ Each player has a board representing their own ships and what 
    they know about their opponent's ships """
    * cell_grid: List[List[CellState]]
    % get_cell -> CellState
    % update
    % ship_sunk -> bool
    % ship_can_occupy -> bool

- Player  ← abstract base class
    * name: str
    * guess_board: Board            ← Opponent's board
    * ships: list[Ship]             ← Player's own ships
    * unguessed: Unguessed          ← Collection of unguessed cells
    % place_ships() -> None         ← abstract
    % take_turn() -> Coordinate     ← abstract
    % output() -> None              ← abstract
    % turn_result() -> None
    % take_hit() -> AttackResult
    % save_move() -> None

    - HumanPlayer(Player)
        % output() -> None    ← Prints to terminal
        % take_turn()         ← Prompts for coordinate guess

        - ClientPlayer(HumanPlayer)
            - conn: TCP connection
            - addr: IP address
            - displayer: Displayer

    - ComputerPlayer(Player)
        * difficulty: Difficulty (enum: EASY | MEDIUM | HARD)
        * mode: Mode (enum: HUNT | TARGET)
        * targets: List[Coordinate]   ← for medium, hard: stack of coords to try after a hit
        * hit_ships: Set[Ship]        ← for medium, hard: keeps track of
                                                          recently hit ships
        * prob_map: Dict[Coordinate : int] ← for hard: number of ships that can
                                                       occupy each cell
        % take_turn()     ← dispatches to strategy
        % turn_result() <- None

- Displayer
    * player_board: Board
    * opp_board: Board
    * ships: List[Ship]
    * print_fn: Function
    % _draw_boards_side_by_side -> None
    % display -> None

    - SimulationDisplayer(Displayer)
        * print_fn: Function (no-op)
        % display() -> None (no-op)

- GameManager
    * p1: Player
    * p2: Player
    * cur_turn: int
    * displayer: Displayer
    % start() -> Player             ← main game loop
    % turn()  -> Tuple[bool, str]   ← each player takes one turn

- BattleshipClient
    - conn: TCP connection
    % start
